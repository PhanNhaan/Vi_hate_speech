## Cài đặt
Tạo file venv
```
    python -m venv .venv
```

Activate venv
```
    .venv/Scripts/activate
```

Cài các requirements
``` 
    pip install -r requirements.txt
```

Xuất thư viện ra file requirements.txt
``` 
    pip freeze > requirements.txt
```

Deactivate venv
```
    deactivate
```
